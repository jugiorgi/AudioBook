## Sobre

O Audiobook é uma plataforma para ouvir os melhores livros

<br />

##### Instalando as dependências

```sh
yarn install
```

##### Rodando o JSON Server para simular uma API

```sh
yarn server
```

##### Rodando a aplicação em desenvolvimento

```sh
yarn dev
```

##### Rodando a aplicação em produção

```sh
yarn build
yarn start
```
